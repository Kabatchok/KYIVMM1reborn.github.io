Правила

Мінімальні правила комʼюніті (можеш змінити текст як хочеш).
  • Без токсичності: образи, цькування, дискримінація — бан.
  • Без спаму: флуд, реклама, самопіар без дозволу — видалення/мут.
  • Без обману: шахрайство, фейкові “роздачі”, скам-лінки — бан.
  • Поважай модерацію: апеляції — в адекватній формі, без “наїздів”.
  • Без NSFW: контент 18+ не постимо.
  • Без обходу покарань: твіни/обхід бану — перманент.
Порада: винеси детальні правила в Discord (канал #rules), а тут залиш короткий “конспект”.
